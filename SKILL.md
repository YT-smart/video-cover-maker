# 视频封面生成

用这个 skill 把用户的标题、文案、对标图或本人形象照转成可执行的封面方案。默认目标是「一眼能懂、强点击欲、强设计感、强网感」，不是干净普通的知识海报。

## 先判断入口

按用户输入选择路径：

1. **文案/标题生成封面**：用户给标题、视频主题、脚本、爆点或一句想表达的观点。
2. **套用默认模板**：用户要求「用默认风格」「来几个模板」「网感一点」「像爆款封面」。
3. **对标复刻**：用户给一张封面图、截图、链接或描述，要求复刻、仿做、参考这个风格。
4. **使用本人照片**：用户上传或指定个人形象照片，或说「用我的照片」「用之前那张照片」。

如果用户没有指定比例，同时输出两版：横屏 `4:3` 和竖屏 `3:4`。如果平台明确，按平台习惯补充：小红书/视频号优先竖屏，B站/YouTube 优先横屏。

## 用户照片库

当用户提供本人照片时，先把照片保存到：

`assets/user-photos/`

命名格式：

`YYYYMMDD-brief-description.ext`

同时在回复里说明已保存的文件名。以后用户说「用我的照片」「用之前的形象照」「用主理人照片」时，优先检查这个目录并使用最近或最匹配的照片。

照片选择与使用细则见 [references/portrait-library.md](references/portrait-library.md)。

## 默认输出

除非用户只要一张成品图，否则输出结构保持简洁：

```markdown
## 封面方向
[一句话说明核心点击点]

## 横屏 4:3
- 主标题：
- 副标题：
- 画面：
- 人物：
- 字体/排版：
- 配色：
- 生图提示词：

## 竖屏 3:4
- 主标题：
- 副标题：
- 画面：
- 人物：
- 字体/排版：
- 配色：
- 生图提示词：

## 可选变体
1. [更冲突/更理性/更故事感的备选方向]
```

如果用户明确要直接生成图片，并且当前环境有图片生成工具，直接调用图片生成工具产出封面，不要只停留在方案。

## 设计原则

- 先提炼「点击理由」：观众为什么要点进来，而不是先讨论装饰风格。
- 标题必须短、狠、可扫读。主标题优先 4-10 个中文字，最多两行。
- 画面只服务一个情绪：冲突、反差、稀缺、震惊、答案感、身份代入、结果展示。
- 人物照片要有清晰面部、眼神、姿态或手势；不要把人物当装饰摆件。
- 字体要大、粗、对比强，留出手机缩略图可读性。
- 避免堆字、堆元素、堆图标；一张封面只表达一个钩子。
- 生成图片时明确比例、构图、文字位置、人物位置、主色、背景层次和不要出现的元素。

## 网感最低标准

默认封面不能只是「整洁、科技、专业」。除非用户明确要求极简高级风，每张封面都要有至少 4 个强网感信号：

- 超大粗体标题，关键词用荧光黄、红色或反色框突出。
- 人物或主体足够大，占画面 35%-55%，能在信息流缩略图里抓眼。
- 强对比色块，优先黑/白/红/黄，少量蓝绿科技光效只做辅助。
- 手绘圈注、箭头、爆炸贴纸、警告框、对勾/叉号等视觉钩子。
- 画面有明确动线：第一眼看主标题，第二眼看人物，第三眼看解释性元素。
- 主题能被视觉化：例如「工具 vs 团队」要画成单个工具被叉掉、多张 skill 卡片连接成工作流。

如果生成结果看起来像普通海报、课程海报或干净 PPT 封面，要主动升级为「写实人像 + 手绘涂鸦 + 综艺贴纸 + 高对比排版」。

## 质量底线

不要生成简陋封面。极简不等于简陋，警告牌风、产品发布风、科技风也必须有完整的设计完成度。

禁止这些低完成度结果：

- 大面积空白背景 + 一个简单图标 + 几个字。
- 像临时 PPT 首屏、普通课程海报、廉价广告图或模板站占位图。
- 只有概念，没有主体、层次、材质、光影、动线和社交平台缩略图冲击力。
- 人物太小、标题太弱、视觉元素没有解释内容卖点。
- 3D 风只有几个悬浮方块，没有明确产品场景、主视觉和发布会质感。

做极简风时，也要具备：强主体符号、精致材质、明确阴影/光影、强排版比例、品牌级留白、可截图传播的视觉记忆点。做产品发布风时，要具备：核心产品主视觉、功能模块层级、环境光、空间深度、人物或场景锚点、像真实发布会海报一样的完成度。

## 默认风格模板

当用户要默认模板或没有明确风格时，读取：

[references/style-templates.md](references/style-templates.md)

从里面选择 1-3 套最合适的模板，不要机械全列。优先选择能放大用户标题冲突或视频卖点的模板。有人物形象照时，默认优先使用「主理人涂鸦爆款封面」或「综艺感情绪爆发」。

## 对标复刻

当用户给对标图时，读取：

[references/reference-replication.md](references/reference-replication.md)

先拆解对标图的布局、标题层级、色彩、主体、情绪和点击机制，再生成「同结构、同网感、不同内容」的新封面。除非用户明确拥有授权，不要照抄水印、Logo、人物身份或完整受保护图像。

## 生成提示词规则

提示词要包含这些信息：

- 画幅：`4:3 horizontal video cover` 或 `3:4 vertical video cover`
- 画面主体：人物、产品、场景或符号
- 情绪：震惊、笃定、悬念、反差、专业、故事感
- 文字区域：标题放在左侧/上方/底部，保留大块干净留白
- 字体气质：粗黑体、综艺感粗字、科技感无衬线、杂志标题
- 色彩：主色、强调色、背景色
- 质感：真实摄影、棚拍、剪贴拼贴、信息图、强对比海报
- 完成度：主体、前景、中景、背景、材质、光影、动线和信息层级都要明确
- 约束：不要小字堆砌、不要复杂背景、不要水印、不要多余 Logo、不要错误文字

如果图片生成模型不擅长准确生成中文文字，优先让画面留出文字区域，然后建议后期用设计工具叠加中文标题。
