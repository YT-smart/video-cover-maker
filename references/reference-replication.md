# 对标封面复刻流程

目标是复刻「点击机制和视觉结构」，不是无脑复制一张图。对标图能提供构图、节奏、字体层级和情绪，但新封面要替换成用户自己的内容、人物和标题。

## 拆解顺序

1. **比例与平台**：横屏、竖屏、方图、是否明显来自 B站/小红书/抖音/视频号。
2. **主体位置**：人物、产品、截图、符号分别占画面多少。
3. **标题层级**：主标题几行、每行字数、关键词是否放大、是否有描边/色块/倾斜。
4. **视觉动线**：观众第一眼看哪里，第二眼看哪里。
5. **点击机制**：冲突、悬念、结果、恐惧、身份代入、答案感、反差。
6. **配色与质感**：主色、强调色、背景复杂度、是否有截图/拼贴/电影感/综艺感。
7. **可复用结构**：提炼成布局公式，例如「左标题右人物」「中间大物体上下双标题」「左右对比」。

## 输出方式

先给一段复刻分析，再给新封面方案：

```markdown
## 对标拆解
- 布局：
- 标题：
- 色彩：
- 主体：
- 点击机制：

## 复刻方案
- 保留：
- 替换：
- 避免：

## 生成提示词
[横屏/竖屏提示词]
```

## 复刻边界

- 可以复刻：构图比例、标题层级、配色气质、情绪表达、信息密度。
- 应该替换：具体人物、Logo、水印、品牌识别、原封面中的专属截图或专属插画。
- 如果用户明确说这是自己的素材或授权素材，可以更接近原图，但仍要优先服务新标题与新内容。
- 不要承诺 100% 像素级复刻；封面应该做到「观感同类、点击逻辑一致、内容属于用户」。

## 复刻提示词骨架

```text
Create a new viral video cover inspired by the provided reference image.
Keep the same composition logic, title hierarchy, color mood, and clickbait energy,
but replace all original subject matter with: [用户主题/人物/标题].
Aspect ratio: [4:3 or 3:4].
Main subject: [人物/产品/场景].
Title area: [位置和留白].
Style: [对标图风格拆解后的描述].
Avoid copying logos, watermarks, exact characters, or protected brand elements.
No clutter, mobile-readable thumbnail, high contrast.
```
